Nastavení Azure
Jednorázové nastavení (~5 min). Potřebuješ zaregistrovat aplikaci v Microsoft Azure.
1 Otevři portal.azure.com a přihlas se svým Microsoft účtem.
2 Vyhledej „App registrations" a klikni na „New registration".
3 Zadej název (např. CCS Tracker), typ účtu: „Accounts in any organizational directory and personal Microsoft accounts", Redirect URI: SPA → http://localhost a pak taky přidej adresu kde máš tuto appku (nebo file://).
4 Po vytvoření zkopíruj Application (client) ID — vypadá jako: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
5 V sekci API permissions přidej: Microsoft Graph → Mail.Read (delegated).
